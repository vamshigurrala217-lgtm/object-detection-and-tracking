# object-detection-and-tracking
Developed a real-time object detection and tracking system using computer vision and deep learning techniques. The system detects multiple objects from live camera feeds or video files, identifies their positions, and continuously tracks their movement across frames. It uses a pre-trained deep learning model for object detection 
Object Detection and Tracking

Object detection is a computer technology related to computer vision and image processing that deals with detecting instances of semantic objects of a certain class (such as humans, buildings, or cars) in digital images and videos.
